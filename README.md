# wechat SDK for golang
https://github.com/chunbaise/wechat

## 简介
| 模块  | 描述                     |
|-----:|:-------------------------|
| mp   | 微信公众平台 SDK           |
| corp | 微信企业号 SDK             |
| mch  | 微信商户平台(微信支付) SDK   |
| oauth  | 用户授权相关   |
| util  | 微信相关工具函数   |

## 安装
go get -u github.com/chunbaise/wechat
